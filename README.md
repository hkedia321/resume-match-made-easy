# Resume Lens

Resume Lens is a platform to review your resume and compare it with the job you want to apply.
Built using Lovable.dev and Cursor.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
